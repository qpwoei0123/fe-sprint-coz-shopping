# requirements

+ Node.js : 18.14.2

+ npm: 9.6.6

# installation
```
$ git clone https://github.com/qpwoei0123/fe-sprint-coz-shopping.git
$ cd fe-sprint-coz-shopping
$ npm install
$ npm run start
```
